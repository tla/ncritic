run_collation.pl - Takes a bunch of texts and returns collation results in 
the specified output format.

see_links.pl - A visualization script; just shows the "similar" vs. 
"different variant" links between the matched words.

do_edit.pl - Takes collation output in TEI parallel-segmentation form and 
presents the user with the editing choices that must be made.  Simple 
STDIN/STDOUT interface.

stemma_data.pl - Collates the requested files and creates a file suitable
for analysis in a PARS package,
e.g. http://mobyle.pasteur.fr/cgi-bin/MobylePortal/portal.py?form=pars
